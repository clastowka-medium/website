# carolchasemedium.com

Personal website for **Carol Chase, M.A.** — Evidential Medium and Certified Grief Educator.

Hand-written static HTML/CSS, hosted on GitHub Pages. No build step. Designed to be edited with the help of an AI assistant (see [CLAUDE.md](CLAUDE.md)).

## Local preview

Just open `index.html` in a browser, or run a tiny local server:

```sh
python3 -m http.server 8000
# then visit http://localhost:8000
```

## Deploying

GitHub Pages serves from `main`. Push to `main` → live in ~30 seconds.

## Editing

If you're an AI assistant helping Carol update the site, read [CLAUDE.md](CLAUDE.md) first — it has the section map, common edit recipes, and tone guidelines.
